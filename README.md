## Usage

```
repositories {
  maven { url 'https://jitpack.io' }
}

dependencies {
  implementation 'com.github.Fabric-Utility-Mods:FriendAPI:VERSION'
}
```

Then call `FriendManager.INSTANCE.init();` when you initialize your mod.
